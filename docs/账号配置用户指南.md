# 账号配置用户指南

这是一份面向日常使用的简版文档，只讲怎么配、怎么用。

## 1. 先理解两套配置

1. 账号配置页：只管理账号身份（账号名/手机号）和账号任务覆盖参数，不填密码。
2. 任务配置页：管理任务本身开关与运行账号列表（这里才填写账号,密码）。

## 2. 快速上手

1. 打开账号配置页，在“账号列表”每行填一个账号名（手机号）。
2. 点击“保存账号列表”。
3. 选择账号和任务，按需修改该账号在该任务下的覆盖参数并保存。
4. 回到对应任务页，填写运行账号列表（账号,密码），并按需开启：
   - 多账户模式
   - 多账户独立配置

## 3. 关键规则

1. 账号名（手机号）是唯一ID。
2. 改密码不会影响账号ID。
3. 改账号名会被视为新账号（新建ID）。

## 4. 常见场景

1. 只想给某个账号单独调参数：
   - 在账号配置页选中该账号和任务，修改后保存。
2. 想多账号轮换跑任务：
   - 在任务页填写账号,密码并开启多账户模式。
3. 想多账号独立配置：
   - 在任务页开启多账户独立配置。
4. 账号配置页看不到某个任务：
   - 该任务当前不支持多账号独立配置展示。

## 5. 常见问题

1. 为什么账号配置页不填密码？
   - 因为账号配置页只做账号身份与覆盖编辑，密码只用于任务实际登录。
2. 我改了账号配置页账号列表，主任务没跟着变？
   - 正常。账号页账号列表与任务页账号列表是独立的。
3. 任务页账号列表格式写错会怎样？
   - 空行、缺少逗号、账号名为空的行会被自动忽略。
   - 当前不会逐行输出格式错误日志，避免刷屏；建议先在账号配置页维护账号名，再在任务页维护账号,密码。
