# 日常任务

### 运行方法

可以在 GUI 中点击『任务/日常任务/开始』以运行日常任务，也可以通过命令行参数实现自动化启动，命令如下：

``` pwsh
# 启动后自动执行第1个任务『日常任务』，并在任务完成后退出程序
ok-ef.exe -t 1 -e
```

* `-t` 或 `--task`: 启动后自动执行第N个任务。`1` 代表任务列表（文件 [./src/config.py](./src/config.py) 列表 `onetime_tasks`）中的第1个（也就是『日常任务』）。
* `-e` 或 `--exit`: 任务执行完毕后自动退出程序。

### 运行顺序

运行顺序定义在文件 [./src/tasks/DailyTask.py](./src/tasks/DailyTask.py) 函数 `run` 列表 `tasks`。简短概括如下： 

1. 送礼：通过「帝江号/干员联络台/赠送礼物」提升员好感度
2. 据点兑换：在「地区建设/据点管理」中通过交易获得地区兑换券
3. 转交运送委托：在「地区建设/仓储结点」中转交全部运送委托
4. 转交委托奖励领取：领取「地区建设/仓储结点/我转交的委托」
5. 造装备：前往「装备制造/套组装备制造」并制作一件列表首位的装备
6. 收信用：前往好友的「帝江号」并在「访客终端」上进行助力获得信用
7. 收集线索：前往「帝江号/会客室」收集全部线索
8. 制造舱：前往「帝江号/制造仓」收取培养材料
9. 买信用商店：在「采购中心/信用交易所」采购「武库配额」「嵌晶玉」
10. 买卖货：在「弹性需求物资」与好友交易赚取调度券
11. 刷体力：消耗所有「理智」刷取培养材料
12. 日常奖励：领取「行动手册/日常」中的奖励

### 送礼

> 检查单：「帝江号/剑桥」可传送，「干员联络台」可用。

通过「帝江号/干员联络台/赠送礼物」提升员好感度。如果途中偶遇干员，则直接交互完成送礼。任务开始时候，角色不能位于「帝江号/剑桥」传送点附近。

角色会传送到「帝江号/剑桥」，然后走向「干员联络台」，最后交互完成送礼。

选项：『送礼』『送礼任务最多尝试次数』『优先送礼对象』

### 据点兑换

> 检查单：「地区建设/据点管理」可用，至少一个据点可交易。

在「地区建设/据点管理」中通过交易获得调度券。

程序会遍历所有地区所有据点，尽可能获得全部调度券。**程序支持的交易货品** 详见文件 [world_map.py](../src/data/world_map.py) 中的列表 `goods_dict` 。

选项：『据点兑换』『优先货品交易序列』

#### 优先货品交易序列

选项『优先货品交易序列』默认为空，此时程序会识别 *据点接受的货品*，对于其中 *程序支持的交易货品* 按照 **随机顺序** 交易。

如果想要程序按顺序进行交易，可以在选项『优先货品交易序列』中 **依次填入货品名称并用英文逗号 `,` 隔开** ，如此以来：

- 对于 **优先货品** ，如果同时属于 *据点接受的货品* 和 *程序支持的交易货品* ，那么按照 **填写顺序** 交易。
- 对于其它货品，在优先货品后按照随机顺序交易。
- 错误的/据点不接受的/程序不支持的 货品名称会被忽略。

#### 例子

游戏版本1.1，四号谷地 `20晶体外壳 + 24柑实罐头 + 18精选荞愈胶囊 + 18高容谷地电池` 毕业满矿产线，武陵 `6优质芽针针剂 + 12中容武陵电池` 版本毕业产线。

选项『优先货品交易序列』可以填入 `晶体外壳,柑实罐头,精选荞愈胶囊,高容谷地电池,优质芽针针剂,中容武陵电池`，这样可以在四号谷地节余电池（搬运发电减少武陵电池消耗）并且在武陵优先消耗药品（防止药品爆仓阻塞污水堵住电池产线）。

### 转交运送委托

> 检查单：「地区建设/仓储结点/委托运送列表」可用。

是否在在「地区建设/仓储结点」中转交全部运送委托。程序会遍历所有区域的「仓储结点」并转交全部委托。

可能出现 委托未领取/货物摔坏 情形，不能确保获得最大收益。

选项：『转交运送委托』

### 转交委托奖励领取

> 检查单：「地区建设/仓储结点/我转交的委托」可用。

是否领取「地区建设/仓储结点/我转交的委托」。超时委托也会被领取。

选项：『转交委托奖励领取』

### 造装备

> 检查单：「装备制造/套组装备制造」可用。

是否前往「装备制造/套组装备制造」并制作一件 **列表首位** 的装备。请确保有足够的 **装备原件** 和调度券。

如果想要长期托管，建议保留一条装备原件产线（每日产量不小于50）。

选项：『造装备』

### 收信用

> 检查单：「帝江号」区域可达。拥有至少25个好友。「采购中心/信用交易所」可用。

是否前往好友的「帝江号」并在「访客终端」上进行助力获得信用。助力结束后，前往「采购中心/信用交易所」收取全部助力。

若选项『尝试仅收培育室』开启，则优先尝试仅助力好友「帝江号」上的「培养仓」。如果不能，至少助力一次其它舱室。

选项：『收信用』『尝试仅收培育室』

### 收集线索

> 检查单：「帝江号/会客室」区域可达。

是否前往「帝江号/会客室」收集全部线索。若集齐线索，则开启情报交流。

如果想要提高情报交流频率（获得更多的信用来兑换抽卡资源），建议多加好友。

选项：『收集线索』

### 制造仓

> 检查单：所有的「帝江号/培养仓」区域可达。每个区域设好待制造物品。每个区域至少安排一名角色。

是否前往「帝江号/制造仓」收取培养材料。收取后会补足待制造数量。

如果想要提高制造效率，请安排天赋合适的角色。

选项：『收集线索』

### 买信用商店

> 检查单：「采购中心/信用交易所」可用。

是否在「采购中心/信用交易所」采购。自动刷新 且 仅购买「武库配额」「嵌晶玉」。

若剩余信用小于选项『信用商店保留信用』，则终止采购。

选项：『买信用商店』『信用商店保留信用』

### 买卖货

> 检查单：「帝江号」区域可达。拥有至少25个好友。「地区建设/物资调度/弹性需求物资」可用。

是否在「弹性需求物资」与好友交易赚取调度券。自动选择利润最高的方案，然后用价格上下限判定是否交易。（除非可购买数量即将溢出，这种情况下必定交易。）

当前 **程序支持的交易物资** 详见文件 [world_map.py](../src/data/world_map.py) 中的列表 `exchange_goods_dict` 。

选项：『买卖货』『武陵买入价』『武陵卖出价』『武陵』『四号谷地买入价』『四号谷地卖出价』『四号谷地』
